### Item58. 전통적인 for문 보다는 for-each문을 사용하라
- 핵심정리
    - 전통적인 for문과 비교했을 때 for-each문은 명료하고, 유연하고, 버그를 예방해준다.   
      성능저하도 없다. 가능한 모든 곳에서 for-each를 사용하자.
